# panda_tutor
PandaTutor — это full‑stack веб‑приложение‑маркетплейс для онлайн‑репетиторства. Платформа позволяет ученикам искать репетиторов по предметам и цене, бронировать и оплачивать занятия, оставлять отзывы и вести переписку в встроенном чате. Репетиторы управляют профилем и расписанием слотов, администратор следит за пользователями, бронированиями.
