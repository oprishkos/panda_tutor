# PandaTutor

PandaTutor — это full‑stack веб‑приложение‑маркетплейс для онлайн‑репетиторства. Платформа позволяет ученикам искать репетиторов по предметам и цене, бронировать и оплачивать занятия, оставлять отзывы и вести переписку во встроенном чате. Репетиторы управляют профилем и расписанием слотов, администратор следит за пользователями и бронированиями.

## 🚀 В планах

В ближайших обновлениях планируется внедрение AI-функциональности:
- **Умный подбор репетитора** — автоматические рекомендации на основе целей и уровня ученика
- **AI-ассистент** — помощник для учеников внутри платформы
- **Анализ прогресса** — отслеживание успеваемости и персональные советы по обучению

## 🐳 Запуск через Docker

### Требования
- [Docker](https://www.docker.com/products/docker-desktop)

### Установка и запуск

1. Клонируй репозиторий
```bash
   git clone https://github.com/oprishkos/panda_tutor.git
   cd panda_tutor
```

2. Создай `.env` файл на основе примера
```bash
   cp .env.example .env
```

3. Собери Docker образ
```bash
   docker build -t panda_tutor .
```

4. Запусти контейнер
```bash
   docker run -p 5000:5000 panda_tutor
```

5. Открой в браузере
```
   http://localhost:5000
```
